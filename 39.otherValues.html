Code needed for this assignment, Part II


One of the cautions when using an input text box is that values inside the textbox are treated as text symbols, even if they contain nothing but number symbols. For example, if you enter 34 into the textbox, instead of having a numerical value of 34, it will think it is the symbols "34" and that it is Not a Number (NaN). We need to convert the symbol into a numerical value, but fortunately there are functions to do that for us. Look for the getNumber() function in our code to see how it can be used.
Note that if the symbols cannot be converted to numbers, they store a value of "NaN" (not a number)
