A table is a great way to organize information in your page for example a class list, or league standings; basically any data that can be organized into a series of rows and columns where you might want to preserve spacing.
To define your table, we use a series of nested elements:
  1. There is a parent element of <table>
  2. There is a first child element to define a row <tr>
  3. Within each row, there are child elements for individual cells <tr>
