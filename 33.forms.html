Forms are what we refer to as elements that can create input. They generally consist of buttons, checkboxes, radio buttons, drop down menus (select boxes), and text input fields. There are also hidden input elements, and elements that will only accept phone numbers, or emails, or dates, but these are just really variations of input fields:

Note that form elements do not have to be container tags. In fact, with new javascript and css, they actually don't even have to be form elements, but we continue to use form tags as they are a good starting point that helps us keep the names and functions associated.
