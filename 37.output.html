How does this work?
  • The document has 2 div tags, one that is identified as "content" and one that is identified as "explanation". Explanation has a style applied to it to make it invisible at the start.
  • An event handler is created to wait for the page to load. The event handler listens for a click event on the button. Once the button is clicked, the function "update" is called
  • In the update function, the target element (the one that we are going to update the content for) is identified. This is the one whose HTML content is going to be changed.
  • We could write the changes directly, but we have a lot of text that we're going to add, so we are going to create a variable to store the changes, and then add onto the variable using the += operator
  • Once we have our content stored in a variable, we'll assign that directly to a property of our ID object, using the "innerHTML" property
  • Each style property is updated directly using variables. Sometimes, you'll need to hunt around to find out what the name of the actual property you want to change is called, but some code editors like Microsoft Visual Studio Code can help you with autocompletion and suggestions.
  • The hidden block has its visibility style changed to make it visible!
