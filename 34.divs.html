Today you will be using styles to create blocks of code that can be positioned around the page. You will be creating a layout for your form or creating a new one for extra practice!

It is a good idea to create a main element that will set the limits of your form. Then all of your new elements and sections can be placed within the limits of this border.

Another good idea is to add borders around all of your elements when you are placing them; they can always be removed later, but it will help you visualize where they are to make sure that you have not made errors in your code.
